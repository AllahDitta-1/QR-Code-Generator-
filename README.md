# QR Code Generator

A simple frontend tool for generating QR codes from user input with a clean browser-based interface.

## Features

- Generates QR codes from typed text or links.
- Runs fully in the browser for quick local use.
- Simple UI designed for utility-focused interaction.
- Good practice project for integrating QR generation into a frontend app.

## Tech Stack

- HTML
- CSS
- JavaScript

## Getting Started

- Clone or download this repository.
- Open index.html in your browser, or run the folder with VS Code Live Server.


## Notes

- If the project depends on an external QR service or library, internet access may be required while testing.
- Open index.html in a browser to use the generator.
